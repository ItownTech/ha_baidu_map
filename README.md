# 百度地图
在HA里使用的百度地图，支持GPS定位轨迹显示

# 使用方式

```
# 使用内置密钥（有限额，最好自己申请密钥）
ha_baidu_map:

# 完整配置
ha_baidu_map:
  name: 百度地图
  icon: mdi:map-marker-radius
  ak: 百度地图AK密钥

```
